# Data Transformation and Feature Engineering

## Overview

This assignment demonstrates essential data preprocessing, transformation, and feature engineering techniques commonly used in machine learning workflows. The objective is to transform raw data into a structured and meaningful format that improves data quality and prepares it for analysis and predictive modeling.

## Features Implemented

* Data Loading and Exploration
* Feature Engineering
* Date and Time Transformation
* Feature Extraction from Datetime Columns
* Data Normalization using Min-Max Scaling
* Data Standardization using Standard Scaler
* One-Hot Encoding for Categorical Features
* Label Encoding for Categorical Variables
* Outlier Detection using the Interquartile Range (IQR) Method

## Dataset

A sample property dataset was used containing information such as:

* Property ID
* Property Price
* Property Size (Square Feet)
* Listing Date
* Year Built
* Property Type
* Location

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn

## Project Workflow

1. Create and load the dataset.
2. Perform feature engineering by generating new attributes.
3. Transform date columns into datetime format.
4. Extract useful temporal features.
5. Apply scaling techniques to numerical features.
6. Encode categorical variables.
7. Detect potential outliers using statistical methods.
8. Prepare the dataset for machine learning applications.

* Data Transformation

## Conclusion

Data transformation and feature engineering play a crucial role in improving dataset quality and enhancing model performance. This project demonstrates the practical implementation of fundamental preprocessing techniques that form the foundation of many machine learning pipelines.
